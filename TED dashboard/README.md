# Построение дашбордов в Tableau

**Задача:** TED (от англ. technology, education, design — «технологии, образование, дизайн») — некоммерческий фонд, который проводит популярные конференции. На них выступают специалисты из разных областей и читают лекции на актуальные социальные, культурные и научные темы. Необходимо исследовать историю TED-конференций и создать дашборд в Tableau на основе полученных данных.

Дашборд, визуализация данных, Tableau.

**Навыки и инструменты:** Tableau, построение дашбордов.
